# 🐍 Snake Game

A classic Snake game built with HTML, CSS, and JavaScript. Play in your browser—no install required.

**[▶️ Play the game](https://jonritzarnibal.github.io/snake-game/)** *(enable GitHub Pages first—see below)*

## Features

- **3 difficulties:** Easy (wrap edges), Normal (die on walls), Hard (random obstacles + walls)
- No time limit—play until you hit yourself, a wall, or an obstacle
- High score saved in your browser
- Responsive layout, keyboard controls (arrows or W A S D)

## How to play

1. Press any **arrow key** or **W A S D** to start.
2. Move the snake to eat the red food. Each food adds 10 points and lengthens the snake.
3. Don’t run into yourself, the walls (Normal/Hard), or the purple obstacles (Hard only).

## Run locally

Open `index.html` in your browser, or use a local server:

```bash
# From the snake-game folder:
npx serve .
# or: python -m http.server 8000
```

## Tech

- Vanilla HTML, CSS, and JavaScript
- Single-file game logic in `index.html`
